Some tools:

that I use and like

Appart from AngularJS, here are some tools I'm currently playing with and that I love:

  • Webpack is one of those tools you can't imagine living without anymore. It builds, lints and bundles your code, transpiles es6 on the go etc in an incredibly efficient fashion. Grunt and Gulp seem so far away ...
  • React is a revolution in the front-end environment. I love server rendering, native mobile development in javascript as well as what it allows once combined with other technologies: hot module replacement, unidirectional dataflow and so on.
  • Redux is a minimalistic library implementing a flux-like data flow in your application. This is a beautiful solution to managing the state of your application. I just LOVE it.
  • Immutable is going very well with the two previous ones. It enforces you to use a unidirectional functionnal flow. I am a fan of the philosophy.
  • ES6 and ES7 with Babel there is no coming back after trying new es6-7 features.
  • Koa is a very interesting Node server based on co.js routine, making great use of new ES6 generators.
  • Bitcoin. Pretty much un-related, but I've been digging into the blockchain a little, and I'm very keen on trying to make some apps with it!
